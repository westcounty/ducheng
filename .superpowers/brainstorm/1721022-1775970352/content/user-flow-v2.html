探索任务 — 修正后的用户流程

以「任务」为核心维度，任务包含多个地点和子任务

数据结构
任务
如「武康路文艺漫步」
→ 包含
地点 × 子任务
📍 地点A → 📸 拍照 🧩 解谜
📍 地点B → 🎯 抵达 📸 拍照
📍 地点C → 🧩 解谜 📸 拍照
子任务按顺序串联，逐步解锁
→ 完成奖励
🖼️🏅
海报 + 徽章
用户流程
1
任务列表
浏览所有任务卡片——标题、封面、难度、预计时长、完成人数
↓ 选择任务
2
任务详情
查看任务介绍、包含的地点列表和子任务概览。点击「开始探索」
↓ 进入任务
3
逐个完成子任务（串联解锁）
子任务1 → 子任务2 → 子任务3 → ...（按顺序，前一个完成才解锁下一个）
📸 拍照 → AI验证 🎯 抵达 → GPS验证 🧩 解谜 → 答案验证 ❓ 问答 → 答案验证
↓ 全部完成
4
获得奖励
自动生成任务海报（用任务中的照片）+ 专属徽章 + 探索记录存档
