三城主题方案

每座城市2个备选主题，点击你偏好的方向（可多选）

🏯 南京

A

「砖上长安」—— 明代城砖密码

核心卖点：南京城墙上每块砖都刻着制造者的名字——这是600年前的"区块链溯源系统"。玩家追踪一块"不该出现的城砖"背后的秘密。

路线：中华门城堡 → 老门东 → 夫子庙/科举博物馆 → 秦淮河 → 瞻园 → 鸡鸣寺/台城 → 玄武湖远眺

终章反转：那块"错误城砖"不是质量问题——是一个工匠故意留下的，砖上的铭文拼在一起是他写给远方妻子的最后一封信。600年后，你是第一个读到这封信的人。

独特机制：城砖铭文辨读、瓮城方位解谜、科举考卷密码、秦淮河灯谜

⭐ 推荐 — 城砖铭文是全国独一无二的物理谜题素材

B

「金陵十二时辰」—— 民国谍战暗线

核心卖点：1946年梅园新村，中共代表团在国民党特务的严密监视下，发展出一套精妙的反监视暗号体系。玩家重走这条"看不见的战线"。

路线：总统府 → 1912街区 → 鸡鸣寺 → 台城城墙 → 六朝博物馆 → 颐和路民国公馆区 → 长江大桥

终章反转：你以为在追踪一份情报，但那份"情报"其实是一张全家福——一个从未曝光的地下党员，用暗号系统传递的不是军事秘密，而是他新生女儿的照片。

独特机制：民国建筑风格辨识、城墙铭文解读、法桐树编号暗号、大桥浮雕计数

🌊 杭州

A

「断桥不断」—— 白蛇传·未删节版

核心卖点：所有人都知道白蛇传的故事，但没人读过"原版"。玩家沿着西湖追踪一份被毁掉的手稿残页，拼出一个与流传版本截然不同的结局。

路线：断桥 → 白堤/孤山 → 西泠印社 → 岳王庙 → 苏堤六桥 → 净慈寺 → 雷峰塔

终章反转：残页拼出的"原版结局"不是白蛇被镇压——而是她自愿走入塔下，因为塔基下藏着的不是镇妖符，而是许仙为她求来的凡人寿命。所谓"千年等一回"，等的不是自由，是衰老。玩家全程拍的西湖照片，恰好是白素贞"凡人一日游"的行程。

独特机制：碑刻拓印解谜、华严经塔八面定位、苏堤六桥首字组合、雷峰塔铜雕排序

⭐ 推荐 — 白蛇传IP全民认知度最高，反转空间巨大

B

「临安密令」—— 南宋末日宝藏

核心卖点：1276年元军兵临城下，一位南宋宫廷画师在亡国前夜将一幅绝世画作拆分藏匿于杭州七处。700年后线索重现。

路线：鼓楼/南宋御街 → 德寿宫遗址 → 胡庆余堂 → 河坊街 → 孤山/浙博 → 西泠印社 → 雷峰塔

终章反转：那幅"绝世画作"不是名画——是画师画的临安城最后一个平静的清晨。你拼出画面的那一刻，发现它画的就是你今天走过的这些地方。

独特机制：南宋考古层识别、"戒欺"匾额方位谜、印章篆刻解读、御街地层年代推理

🐫 西安

A

「长安译」—— 丝路七种文字

核心卖点：大唐长安是当时世界上最国际化的城市——波斯商人、印度僧侣、罗马传教士在此汇聚。玩家追踪一位粟特商人的密账，需要在碑林的叙利亚文、清真寺的阿拉伯文、大雁塔的梵文中破译线索。

路线：小雁塔/西安博物院 → 南门城墙 → 碑林（景教碑） → 书院门 → 钟鼓楼 → 化觉巷清真大寺 → 大雁塔

终章反转：粟特商人的"密账"不是生意账目——是一本用七种文字写的日记。每种文字记录他在长安结交的一个朋友。他用不同的语言，是因为每个朋友只懂自己的母语。这座城市的伟大不在于城墙多高，而在于——一个外国人可以在这里拥有七种语言的友谊。

独特机制：多文字辨识（小篆/楷书/草书/叙利亚文/阿拉伯文）、城砖铭文、二十四节气鼓排序、塔层数学

⭐ 推荐 — 碑林多语言碑刻是全球独一无二的解谜素材

B

「大唐诗案」—— 消失的第三十七首

核心卖点：碑林中藏着一首从未被收录的唐诗——它被一位诗人用密码拆散，嵌入了长安七处碑刻之中。玩家用唐诗格律（平仄、韵脚、对仗）作为密码系统来还原这首诗。

路线：大雁塔/大慈恩寺 → 大唐不夜城 → 曲江池 → 小雁塔 → 碑林 → 书院门 → 城墙南门

终章反转：还原出的诗不是什么名篇——是一个无名诗人写给长安城的告别诗。他离开长安前，把这首诗藏在这座城市里，"如果有人能找到它，说明长安还在。"你今天找到了。

独特机制：诗词格律解密、书法字体辨识（颜体/草书/隶书）、碑文坐标定位、城墙门名字谜

暗线关联设计（三城彩蛋）：三个游戏表面完全独立，但都有一个共同的隐藏元素——每个游戏的终章密文中，都嵌入了另外两座城市的一句话。玩三个城市的玩家最终会发现：三个不同时代、不同城市的主角，留下的最后一句话拼在一起，是同一个意思。
