## 第0站：校对密码（序章，在家完成）

> 不需要到任何地点。用道具解出第一个目的地。

**屏幕显示：**

```
翻开笔记本
━━━━━━━
退稿信:
  一封1936年的退稿信，某些字被校对符号标注
  （○、△、▽、□ 等）

校对符号密码表（"苏蕊的规矩"）:
  ○ = 取该字声母
  △ = 取该字韵母
  ▽ = 取声调数字
  □ = 取该字笔画数
  ……

密码表底部批注:
  "按老规矩，从第█个标记起读——数数我书桌上那摞书。"

书桌旧照片:
  夹在笔记本里的一张旧照片
```

**指令：**
> 退稿信上有些字被校对符号标注了——看起来像正常改稿，但你觉得不太对。
>
> 笔记本里有一张"苏蕊的规矩"——校对符号密码表。每种符号代表一种提取规则。
>
> 密码表底部写着"从第█个标记起读——数数我书桌上那摞书"。旧照片里，苏蕊书桌上摞着几本书？
>
> 找到起始位置，用密码表逐个转换退稿信上被标记的字。拼出来的就是你的第一个目的地。

**解谜过程：**
1. 看书桌旧照片 → 桌上摞着 **N** 本书（`__FIELD__`，道具制作时确定）→ N 是起始位置
2. 从退稿信第 N 个标记起，按符号规则逐字提取：
   - ○标记的字取声母，△标记的字取韵母，▽标记的字取声调数字……
3. 拼出结果 → **福州路**（或"杏花楼"，具体拼法取决于退稿信设计）

**答案：** 福州路 / 杏花楼（具体待道具定稿）
**提示：**
1. 先看旧照片——数桌上的书，那个数字就是起始位置。
2. 从退稿信第N个被标记的字开始，每个符号有不同的提取规则——○取声母，△取韵母……逐个提取。
3. 把提取出的声母、韵母、数字拼在一起，读出来——那是一条路的名字。

**通过后显示：** "福州路。苏蕊在这条街上走了五年。现在轮到你了。" → [前往第一站]
