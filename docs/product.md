# 产品说明

## 产品定位与目标用户

**读城**是一款城市实景解谜游戏平台。玩家在真实城市中沿预设路线行走，在每个地点观察建筑细节、使用纸质道具解谜，最终拼合一个跨越时空的故事。

**核心价值主张：** 让人重新"看"一座城市。不是旅游打卡，不是历史课堂，是用一个虚构角色的眼睛重新走一遍你以为熟悉的街道。

**目标用户：**
- 朋友聚会（3-5 人组队最佳）
- 城市本地青年，想用新方式重新认识自己的城市
- 情侣约会，文化深度游
- 外地游客，想体验不同于打卡的城市探索

## 用户旅程

```
1. 打开 H5 链接 → 城市选择首页
2. 选择一座城市 → 阅读故事开场白
3. 「接受任务」→ 进入第 1 站
4. 每站流程：
   ├── 阅读日记/线索
   ├── 到达实地地点
   ├── 观察 + 道具操作 → 解答 2-4 个步骤
   ├── 拍照存档
   └── 获得密电碎片 → 过渡线索引导至下一站
5. 第 7 站完成 → 进入终章
6. 终章：藏头诗揭示 → 打开信封 → 心愿清单+照片回顾
7. 档案室：回顾全程、展开日记、查看暗线
8. （可选）通关更多城市 → 解锁跨城暗线 → 全通关揭示
```

## 五城剧本概要

### 上海「第七封密电」
- **主角**：1943 年间谍"鹤影"
- **序章**：在家解密烧毁纸条 + 凯撒密码 → 推出第一站"和平饭店"
- **路线**：和平饭店→海关大楼→汇丰银行→豫园九曲桥→一大会址→思南公馆→武康大楼
- **谜题风格**（V3）：隐喻推理型——日记用间谍手记暗示观察目标（"建筑的签名""沉默的巨人""飞檐的守卫"），玩家自行推理后观察
- **终极密电**：「此生未能再见，但你走过的每一条路，我都提前走过一遍。」
- **藏头**：你走过的每一步
- **Meta 反转**：密电不是军事情报，是遗愿清单

### 上海2「墨迹」
- **主角**：1936 年世界书局女编辑苏蕊，暗中编辑被审查删除段落的《墨迹集》
- **区域**：福州路（中华文化第一街）+ 河南中路拐角 + 南京路步行街，全程步行约 2km
- **序章**：校对符号密码——退稿信上的校对标记 + 密码表 + 书桌旧照片互文推理，得出"福州路"
- **路线**：杏花楼→外文书店/世界书局旧址→古籍书店→笔墨博物馆→商务印书馆旧址→百新书局→永安百货
- **谜题风格**：多感官沉浸——每站绑定一个主感官（味觉/嗅觉→视觉→嗅觉→触觉→听觉→综合→仪式），感官体验本身是谜题的一环而非背景
- **日记文体变奏**：7 站 7 种文体（记账式→批注体→古文体→书信体→新闻报道体→口语体→遗书体）
- **终极碎片**：「书不会消失，它只是在等一个人重新翻开。你就是那个人。」
- **藏头**：字里行间都是你
- **Meta 反转**：信封里的"编辑审稿单"——玩家发现自己什么时候变成了苏蕊手下的作者

### 南京「金陵刻痕」
- **主角**：明代验砖吏陆鸣远
- **路线**：中华门→老门东→科举博物馆→秦淮河→鸡鸣寺→台城→玄武湖
- **终极碎片**：「将军刻功，诗人刻诗……七个认真活过的普通人的名字。」
- **藏头**：城巷有声音尚在

### 杭州「断桥不断」
- **主角**：白素贞（蛇的视角）
- **路线**：断桥→孤山/西泠印社→龙井村→岳王庙→苏堤六桥→净慈寺→雷峰塔
- **终极手稿**：「我见过一千年的西湖。但值得留下来的理由，只需要一个下午就够了。」
- **藏头**：湖心有一人未走

### 西安「长安译」
- **主角**：唐代粟特商人康安
- **路线**：小雁塔→南门→碑林→书院门→钟鼓楼→回民街→含光门
- **终极请帖**：「如果你读到了全部七种文字，请来赴宴。等了一千三百年，终于凑齐一桌人。」
- **藏头**：此城容得下万邦

### 苏州「姑苏折子」
- **主角**：清代旦角沈云筝
- **路线**：盘门·吴门桥→苏州文庙→双塔→耦园外墙→平江路→桃花坞→山塘街·星桥
- **终极戏票**：「台上的人散了……你走进来，就算有人来过。」
- **藏头**：借景入梦戏如真
