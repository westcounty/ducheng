# 「金陵刻痕」——南京实景解谜游戏设计文档

> 设计日期：2026-04-02

---

## 1. 概述

### 游戏定位

一款线下实景解谜游戏。南京是一座被无数人刻过字的城市——砖上有工匠的名字，考场墙上有考生的抓痕，秦淮灯笼上有才子的诗句，寺庙钟上有信众的铭文，台城柳下有诗人的叹息。玩家追踪一个"读了整座城"的人留下的七处刻痕。

### 核心参数

| 项目 | 说明 |
|------|------|
| 目标玩家 | 本地年轻人朋友聚会（熟悉南京，追求新奇和深度） |
| 人数 | 2-5人一组 |
| 时长 | 4-5小时（含解谜、拍照、休息） |
| 区域 | 中华门 → 老城南 → 夫子庙 → 秦淮河 → 鸡鸣寺 → 台城 → 玄武湖 |
| 载体 | H5网页 + 可自行打印的实体道具包 |
| 成本 | 玩家端零成本（A4纸打印 + 生活用品替代专业道具） |

### 设计原则

1. **场景绑定**：每个谜题必须依赖该地点的真实物理特征，不可在其他任何地方完成
2. **机制不重复**：7个关卡使用7种不同的核心解谜机制
3. **连环深度**：每个地点包含3-4个串联子谜题，环环相扣
4. **叙事驱动**：谜题不是孤立的智力题，而是叙事推进的手段——每一关的谜题格式模仿一种"南京人说话的方式"
5. **低成本高质感**：所有道具可用A4纸打印+日常物品（手机、纸尺）替代
6. **七种刻字**：每关谜题的格式对应一种历史上真实的"刻字人"身份，与叙事一体

## 2. 叙事框架

### 主线故事

> 陆鸣远，江西人，因识字被征调到南京当验砖吏。本来只需检查每块砖上的铭文是否合规——他的工作就是读砖。但一个识字的人到了南京，怎么可能只读砖？他开始读所有的东西。城门上的匾额、贡院号舍墙上考生偷刻的小字、秦淮河画舫上飘出的半阙词、鸡鸣寺台阶上磨出的经文、台城城头上前朝留下的箭痕。他读了整座城市。
>
> 他死前，在七个地方各留了一处刻痕。不是同一种格式，而是模仿了七种不同的"刻字人"——用砖铭的格式、用木匠落款的格式、用试帖诗的格式、用灯谜的格式、用佛偈的格式、用城头题记的格式、用户籍档案的格式……他把自己的遗言分散藏进了这座城市最不缺文字的七个角落。

### 真实结局（终章揭晓）

陆鸣远留下的不是密码——他只是在每种"南京人说话的方式"里，都加了一句自己的话。将军刻功，诗人刻诗，考生刻愿，僧人刻经，妻子刻平安。他谁都不是，就刻了七个认真活过的普通人的名字。

### 完整拼合语（7段碎片拼出）

> "将军刻功，诗人刻诗，考生刻愿，僧人刻经，妻子刻平安。我谁都不是，就刻了这座城里最不值钱的东西——七个认真活过的普通人的名字。你现在知道了。够了。"

### 日记首字藏字（Meta层）

7篇验砖吏札记的首字连读为：**「城巷有声音尚在」**

含义："城市的街巷是有声音的，那些声音还在。"

## 3. 路线设计

### 总体路线

南→北纵贯，从明城墙南端到玄武湖北岸，纵穿整座金陵城。

```
中华门城堡（起点）
     │  （步行10分钟，沿城墙根向北）
     ▼
  老门东
     │  （步行12分钟，穿过老城南街巷）
     ▼
  夫子庙/科举博物馆
     │  （步行6分钟，沿贡院街）
     ▼
  瞻园/秦淮河
     │  （地铁1号线中华门→鼓楼，约15分钟）
     ▼
  鸡鸣寺
     │  （步行5分钟，登城墙入口）
     ▼
  台城城墙
     │  （步行15分钟，沿城墙至解放门下城）
     ▼
  玄武湖畔（终点）
```

### 七站概览

| # | 地点 | 区域 | 到达方式 | 文化亮点 |
|---|------|------|----------|----------|
| 1 | 中华门城堡 | 城南 | 起点 | 世界最大城堡式瓮城，明代城砖实名制 |
| 2 | 老门东 | 老城南 | 步行10分钟 | 明清民居群落，巷名典故，市井烟火 |
| 3 | 科举博物馆/江南贡院 | 夫子庙 | 步行12分钟 | 中国最大科举考场，号舍、明远楼 |
| 4 | 瞻园/秦淮河 | 秦淮 | 步行6分钟 | 南京最古老明代园林，秦淮灯会 |
| 5 | 鸡鸣寺 | 城北 | 地铁15分钟 | 南朝第一寺，药师佛塔，始建于西晋 |
| 6 | 台城城墙 | 玄武湖畔 | 步行5分钟 | 六朝宫城遗址，"无情最是台城柳" |
| 7 | 玄武湖畔 | 玄武湖 | 步行15分钟 | 六朝皇家园林，明代黄册库 |

### 节奏曲线

- **第1-3关（老城南）**：密集连续，步行距离短，热身+建立信心——从城墙到贡院，沉浸在明代南京的密度里
- **第4关（秦淮河）**：诗意间奏，节奏放缓——从考场的严酷转入园林的婉约
- **地铁跳转**：呼吸时间，队伍讨论整理线索，从城南一跃到城北
- **第5-6关（鸡鸣寺→台城）**：城墙高潮区——登高望远，视野从巷弄骤然打开，难度递增
- **第7关（玄武湖畔）**：湖畔终章，情绪收束——从城墙下来，走到湖边，一切归于平静

## 5. H5功能设计

### 核心功能

| 功能 | 说明 |
|------|------|
| 答案验证 | 每个子步骤/关卡入口输入答案，正确后解锁下一步 |
| 叙事推送 | 每关开始时推送叙事引入文字（配合验砖吏札记） |
| 导航提示 | 解锁下一地点后显示步行/地铁路线（不直接告知地点名，而是用陆鸣远的线索暗示，玩家确认后显示导航） |
| 刻痕存档（拍照） | 每关完成后要求玩家拍摄特定场景照片，存入「刻痕集」 |
| 提示系统 | 每个子步骤可请求提示（分3级：轻推、半解、直接给答案），使用提示会记录 |
| 计时器 | 可选，记录每关用时，通关后显示总成绩 |
| 终章触发 | 第7关完成后，自动进入终章流程 |

### 刻痕存档（拍照）的设计

这是终章Meta揭晓的关键铺垫。在验砖吏的叙事框架下自然融入——陆鸣远读了整座城，玩家也在一站一站地"读"。

| 关 | H5拍照指令 | 实际拍摄内容 | 对应七人名单 |
|----|-----------|-------------|-------------|
| 1 | "拍一块砖——上面有个人的名字，600年了" | 中华门城砖铭文特写 | 城砖工匠名 |
| 2 | "拍一条最窄的巷子——巷子越窄，邻里越近" | 老门东窄巷 | 门楣木匠名 |
| 3 | "拍下这间小屋——曾有人在1.3米宽的空间里写下改变命运的文章" | 科举博物馆号舍 | 落第考生名 |
| 4 | "拍河面的倒影——杜牧看到的秦淮，和你看到的是同一条河" | 秦淮河水面倒影 | 秦淮灯谜人名 |
| 5 | "拍下佛塔和天空——这座塔看过1700年的云" | 鸡鸣寺药师佛塔 | 还愿香客名 |
| 6 | "站在城头面朝玄武湖拍——六朝的皇帝看到的，和你一样" | 台城城墙上俯瞰玄武湖 | 修城民夫名 |
| 7 | "面朝湖面拍——你身后就是这座城六百年的故事" | 玄武湖湖面 | 黄册档案中的普通户主名 |

## 6. 谜题详细设计

### 第1关：中华门城堡——「砖上之名」

**刻痕格式：砖铭（伪装成标准城砖铭文）**

**核心机制：空间观察 + 索引检索 + 铭文解读 + 碎片解锁**

**验砖吏札记首字：城**

**文化层：** 明代城砖质量追溯体系——每块砖刻产地、监造官、工匠名，600年前的"实名制"。朱元璋修南京城墙用砖超过3亿块，每一块都可追溯到具体的人。

**札记引入：**
> "**城**墙下来了新砖。我逐块验过，产地、监造、窑匠，三栏俱全方可入墙。这是规矩。但今日有一块……三栏文字对不上任何府县。我翻遍了名册也查不到。莫非是谁混进了一块不属于这座城的砖？"

**Step 1「瓮城空间推理」**

H5指令：
> "你现在站在中华门城堡入口。走进去，穿过第一道门，进入第二道瓮城。站在正中央，环顾四周——前、后、左、右，各能看到几个门洞？四个数字，就是你的第一组密码。"

玩家行动：进入中华门三道瓮城，站在第二道瓮城中央。前方（向内）看到 `__FIELD__` 个门洞，后方（向外）`__FIELD__` 个，左侧 `__FIELD__` 个，右侧 `__FIELD__` 个。四个数字组成密码A。

得到：密码A（四位数字序列）

进入下一步：H5验证密码A正确后，解锁Step 2指令。

**Step 2「城砖铭文搜索」**

H5指令：
> "用密码A查城砖索引卡——找到对应的那条指令。"

玩家行动：在道具"城砖索引卡"上，用密码A查到一条指令，内容类似：
> "从瓮城 `__FIELD__` 侧墙段，自 `__FIELD__` 门洞起向 `__FIELD__` 方向数第 `__FIELD__` 块砖。这块砖的铭文格式'不对'——产地栏写的不是真实地名。"

玩家在指定墙段找到这块特殊城砖。（注：此砖为游戏虚构设定，实际以道具上的"复刻城砖拓片"替代——道具上印有这块砖的铭文内容，玩家在实地感受城砖铭文系统后，用道具完成后续解读。）

得到：一块"格式不对"的城砖铭文（道具上呈现）

进入下一步：找到铭文后，H5提示进入Step 3。

**Step 3「铭文三栏解读」**

H5指令：
> "这块砖的铭文分三栏：产地、监造官、工匠。每栏取第一个字——三个字连起来，就是第一段碎片的密钥。"

玩家行动：从道具上的城砖铭文中，读出三栏文字，各取首字，组合成一个三字词/短语（例如：`__FIELD__`）。这就是密钥。

得到：三字密钥

进入下一步：在H5中输入密钥。

**Step 4「解锁碎片」**

H5指令：
> "密钥正确。陆鸣远的第一处刻痕——伪装成城砖铭文的遗言——现在属于你了。"

玩家行动：H5显示第1段碎片文字。玩家将碎片抄写到第1张碎片卡上。

得到：**碎片第1段** + 过渡线索

过渡线索（H5显示）：
> "巷深处，门楣上，木匠在百年前落了款。"

**刻痕存档：** "拍一块砖——上面有个人的名字，600年了。"

### 第2关：老门东——「门上之痕」

**刻痕格式：木匠落款（伪装成门框上的工匠签名）**

**核心机制：灯谜猜解 + 实地确认 + 建筑辨识 + 落款搜索**

**验砖吏札记首字：巷**

**文化层：** 南京老城南民居文化——门楣雕刻、巷名典故、市井烟火气。老门东是南京保存最完好的明清街巷，巷名本身就是一部手工业史：箍桶巷、剪子巷、边营……

**札记引入：**
> "**巷**子深处有户人家换门框，我帮着搬旧木头。木匠临走前在门框暗处刻了个名字——他说这是规矩，'将来这扇门坏了，后人知道找谁修。'我想，这和我验砖是一回事。砖上刻名，门上刻名，都是怕被忘记。"

**Step 1「巷名灯谜」**

H5指令：
> "陆鸣远在札记里留了三条灯谜。猜出谜底——它们是老门东三条真实巷子的名字。到了实地，找到巷名路牌，拍照确认。"

道具（札记中）给出三条谜面：
- 谜面一：`__FIELD__`（谜底：箍桶巷 `__FIELD__`）
- 谜面二：`__FIELD__`（谜底：剪子巷 `__FIELD__`）
- 谜面三：`__FIELD__`（谜底：边营 `__FIELD__`）

玩家行动：猜出三条巷名，在老门东街区步行找到对应巷名路牌，逐一拍照确认。

得到：三条巷名确认

进入下一步：H5验证三个巷名后解锁Step 2。

**Step 2「建筑年代辨识」**

H5指令：
> "沿 `__FIELD__` 路线走。看路两侧民居的门楣雕刻——明代的简朴莲花纹、清代的繁复福禄寿、民国的西化几何线条。找到三扇不同朝代的门楣，按时间从早到晚排列它们的门牌号。这个序列就是密码B。"

玩家行动：沿指定路线步行观察。辨识出三种不同时代的门楣雕刻风格：
- 明代门楣（简朴莲花纹）：门牌号 `__FIELD__`
- 清代门楣（繁复福禄寿）：门牌号 `__FIELD__`
- 民国门楣（西化几何线条）：门牌号 `__FIELD__`

按明→清→民国排列门牌号，组成密码B。

得到：密码B（三个数字序列）

进入下一步：H5验证密码B后解锁Step 3。

**Step 3「木匠落款」**

H5指令：
> "密码B对应 `__FIELD__` 号老宅。走到那扇门前，仔细看门框——陆鸣远在这里伪装成木匠，留了自己的落款。"

玩家行动：找到指定老宅门框。（注：实际操作中，道具册提供该门框的"木匠落款拓片"——一段伪装成工匠签名格式的文字，玩家在实地感受门楣文化后，用道具完成解读。）

落款内容 = **碎片第2段** + 过渡线索

过渡线索：
> "天下英才尽入此门，但这扇门只有一米三宽。"

得到：**碎片第2段** + 指向科举博物馆号舍的线索

**刻痕存档：** "拍一条最窄的巷子——巷子越窄，邻里越近。"

### 第3关：科举博物馆/江南贡院——「墙上之愿」

**刻痕格式：考生涂鸦（伪装成号舍墙上的刻字）**

**核心机制：文本格式解码 + 建筑定位 + 碑文检索 + 编号对应**

**验砖吏札记首字：有**

**文化层：** 科举制度的壮阔与残酷——南北榜案（1397年朱元璋亲手干预的科场大案）、号舍中关三天写八股文的极限体验。江南贡院是中国历史上规模最大的科举考场，鼎盛时有号舍两万余间。

**札记引入：**
> "**有**人在号舍墙上刻了字。不是诗，不是经，是一句'娘，儿必中'。字刻得很深，指甲断了也不停。我是来验砖的，不该管墙上的字。但我在那四个字前面站了很久。后来我也在墙角刻了一行——不是给谁看的，就是想在这里留点什么。"

**Step 1「八股文格式密码」**

H5指令：
> "道具中有一篇'残卷'。这是一篇标准的八股文，分成八段：破题、承题、起讲、入题、起股、中股、后股、束股。从每段中提取第N个字——N等于该段在八股中的序号（破题取第1字，承题取第2字……束股取第8字）。八个字连起来，就是你的指引。"

玩家行动：取出道具"八股残卷"，按八股文格式分成8段：
1. 破题 → 取第1个字
2. 承题 → 取第2个字
3. 起讲 → 取第3个字
4. 入题 → 取第4个字
5. 起股 → 取第5个字
6. 中股 → 取第6个字
7. 后股 → 取第7个字

## 8. 验砖吏札记全文与暗线

### 日记暗线：陆鸣远的"读城"

7篇札记中各埋一句关于"读到某种声音/痕迹"的感叹，写得像验砖吏的日常观察，初读不会在意：

| 关 | 札记句子 |
|----|---------|
| 1 | "城墙下来了新砖。每块砖上都有人的名字，但没有人会记住他们。" |
| 2 | "巷子深处的门框上刻着木匠的名字。将来门坏了，后人知道找谁修。" |
| 3 | "有人在号舍墙上刻了'娘，儿必中'。字刻得很深，指甲断了也不停。" |
| 4 | "声从水上来。秦淮河的画舫经过时，总有人在船上唱曲子。" |
| 5 | "音从塔顶来。鸡鸣寺的钟声，从西晋响到现在，石阶上都磨出了经文。" |
| 6 | "尚能登高。台城城头的风比城下大得多，柳树把所有人都送走了还在。" |
| 7 | "在湖边坐了很久。黄册库里全天下人的名字都在，但没有人来读。" |

### 跨城暗线

札记中有一句不起眼的跨城提及：

> "听说杭州西湖边也在修堤，不知那边的泥巴里有没有人偷偷刻字。"

此句在南京游戏中完全自然——一个验砖吏听到同行消息的闲聊。但玩过杭州线路（如有后续开发）的人会注意到这条暗线连接。
