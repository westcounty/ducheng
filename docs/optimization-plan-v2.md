## 1. 西安站7：避免回头路（大雁塔→替代方案）

### 问题
站7大雁塔在城南，需从城中心（回民街）地铁折返25分钟。玩家体力已大，折返体验差。

### 解决方案：将终点改为「含光门遗址博物馆」

**新站7: 含光门遗址博物馆 / 唐城墙遗址公园**
- 地址: 西安市碑林区含光路，含光门内
- 距站6(回民街): 步行约15分钟向南，或骑共享单车8分钟
- **无需地铁，无回头路** — 从钟鼓楼区域自然向南步行即到

**为什么选含光门：**
1. 含光门是唐长安城皇城南面偏西门，遗址保存了**唐/宋/明三代城墙断面**——与站2城墙主题呼应，形成"从外到内"的建筑考古闭环
2. 博物馆内展示了唐代过关文牒（通行证）实物——**完美呼应商人康安的"赴宴请帖"叙事**
3. 傍晚可在唐城墙遗址公园的城墙步道上结束——有日落氛围

**叙事调整：**
- 原设计: 康安的第7位朋友「胡禄」是火祆教徒，关联大雁塔
- 新设计: 「胡禄」改为关联含光门——他是一个通过含光门出入长安的波斯商人，总是拿着通关文牒。康安说"胡禄每次进城都被盘问最久，因为他的货物里总有火焰的气味"
- 终章场景: 玩家站在唐城墙遗址断面前，看到唐/宋/明三层叠加——1300年的时间具象化为眼前的三层土壤。"康安的宴席就摆在这道墙的唐代那一层。你站在明代的层面上，往下看了一眼。"

**新谜题设计（3步）：**

**7-1 (城墙断面·时间分层)**
- 观察含光门博物馆内的唐/宋/明三代城墙断面
- 道具卡上标注了3种不同颜色土层对应的朝代
- 问题: "唐代夯土层位于第几层？"（从上到下: 明1→宋2→唐3，答案=3）
- 提示: 轻推→"最下面的最古老" / 半解→"三层从上到下是明宋唐" / 直接→"答案是3"

**7-2 (过关文牒·七种文字)**
- 展厅内有唐代过关文牒复制品展示
- 道具卡上印有7种文字的"赴宴请帖"片段（梵文/蒙古文/叙利亚文/中文/日文/阿拉伯文/波斯文）
- 问题: 观察展厅里的过关文牒，找到其中使用的文字种类数
- 答案: `__FIELD__`（需实地确认）

**7-3 (碎片拼装·终章)**
- 拿出全部7张密电碎片卡
- 按卡背编号1-7排列
- 拼合后读出完整密文
- 确认类型: confirm（无需精确答案）
- 答案已确认: 完整邀请 "如果你读到了全部七种文字，请来赴宴。我已经备好了酒。等了一千三百年，终于凑齐一桌人。"

**路线对比：**
```
原路线: 回民街(站6) →→ 地铁25分钟 →→ 大雁塔(站7)  ❌回头路
新路线: 回民街(站6) → 步行15分钟 → 含光门(站7)     ✅自然南行
```

## 2. 苏州站5：平江路谜题重设计（颜色匹配→建筑构件匹配）

### 问题
7种传统色（青灰/桃红/粉白/鹅黄/明黄/朱红/月白）受季节/光线/主观性影响太大。

### 解决方案：改为「七扇门」谜题 — 匹配门的形制而非颜色

**核心概念：** 平江路老街有各种不同形制的门——圆门/方门/拱门/月洞门/花窗门/石库门/板门。每种门在昆曲中对应一种"出场方式"（上场门/下场门/将门/闺门/寺门/店门/后门）。

**新谜题设计（3步）：**

**5-1 (七扇门寻访)**
- 指令: "沿平江路从悬桥巷口向北走，找到七种不同形制的门"
- 道具: 7张「门形卡」——每张印有一种门的轮廓线条图（简笔画）+ 昆曲角色名
  - 卡1: 圆洞门 → 杜丽娘（闺阁女子从圆门入场）
  - 卡2: 石库门 → 赵五娘（民间妇人从石库门出）
  - 卡3: 板门（木板简门）→ 陈妙常（尼姑庵的素朴门）
  - 卡4: 拱券门 → 崔莺莺（寺庙的拱门）
  - 卡5: 花窗门 → 杨玉环（宫廷花窗）
  - 卡6: 店铺门（可拆卸门板）→ 白素贞（药店门面）[跨城彩蛋——杭州白蛇]
  - 卡7: 月洞门 → 沈云筝自己（园林月门——她"偷来"的那扇门）
- 任务: 在平江路上找到与每张卡匹配的真实门，拍照记录位置
- 答案: 找到第7种门（月洞门）的门牌号或巷名 = `__FIELD__`

**5-2 (出场序排列)**
- 昆曲有严格的出场规矩: 主角从"上场门"入，配角从"下场门"入
- 参照「折子戏目录」道具，找到这7部戏中角色的出场顺序
- 按出场先后排列7张门形卡
- 排列后，卡背面的数字连起来 = 密码序列
- 答案: `__FIELD__`（需确认7部戏的出场序）

**5-3 (桂花糕暗语)**
- 指令: "替我在平江路买一块桂花糕——这是戏班散场后的规矩"
- 道具「戏班账册」第5页注有: "散场桂花糕，每人一块，记在XX页"
- 翻到对应页: 有一行被墨水涂抹的文字，但首尾两字可见
- 首尾两字 + 门形卡第7张背面的字 = 三字密码 = 碎片解锁
- 答案: `__FIELD__`

**为什么更好：**
- 门的形制是**建筑固定特征**，不受季节/光线/天气影响
- 平江路确实有各种历史门洞，是苏州老城特色
- 与昆曲"出场门"概念关联更紧密
- 保留了"替我买桂花糕"的温暖细节
- 7张门形卡比7张色卡更有辨识度和收藏价值

## 3. 杭州站6：净慈寺谜题优化（增加外部锚点）

### 问题
济公谜语"抬头可见，低头可得"答案开放性过高，可以是"空""响""钟"等多个答案。

### 解决方案：将谜底锚定到净慈寺门口的具体可见物

**优化设计：**

**6-1 保持不变** — "南屏晚钟"三段描述找错误

**6-2 优化 — 锚定到「康熙御碑」**
净慈寺门口有康熙帝题写的"净慈禅寺"匾额，寺内有康熙御碑亭。

新谜语设计（保留济公风格但指向明确）:
> "济公笑着说：'我出一个字给你猜——  
> 它不在佛前，不在佛后；  
> 不在殿上，不在殿下。  
> 抬头，在门楣上见得到它；  
> 低头，在石碑上也找得到它。  
> 这个字，连皇帝都亲手写过。'"

**答案: "净"字** — 既在门楣匾额"净慈禅寺"上（抬头可见），也在寺内碑刻上（低头可见），而且是康熙皇帝亲笔。这个答案有且仅有一个，因为：
- "净"是匾额4字中唯一同时出现在门楣和碑文中的关键字
- "连皇帝都亲手写过"明确指向康熙御笔
- 提示系统可以逐步缩小: 轻推→"看看寺门上方的匾额" / 半解→"四个字中有一个字在碑上也出现了" / 直接→"答案是'净'"

验证任务: 找到"净"字，记录它在匾额上的位置（第几个字）= 答案1

**6-3 优化 — 偈语排序锚点更明确**
- 4首偈语重新设计，每首与净慈寺一个可见元素对应:
  1. 与"钟"相关的偈 → 对应钟楼
  2. 与"净"相关的偈 → 对应匾额（6-2答案确认）
  3. 与"水"相关的偈 → 对应西湖
  4. 与"塔"相关的偈 → 对应远处雷峰塔
- 排序规则: 站在寺门口，按视线从左到右的顺序排列
- 这样排序有唯一正解（钟楼在左，匾额居中偏左，西湖在前方，雷峰塔在右）

**为什么更好：**
- "净"字答案唯一且可验证
- 谜语保留了济公的风趣语气
- "连皇帝都亲手写过"是强锚点，大幅缩小答案范围
- 偈语排序也有了物理参照（视线方向），不再纯靠文意推断
- 需实地确认: 康熙匾额是否仍在？碑亭是否可进入？

## 4. 上海站6：思南公馆卡登格栅可行性重评 & 备选方案

### 研究结论
- 思南公馆有文保标牌，但文字量太少（约20-50字），不足以支撑卡登格栅
- 周公馆铜牌只有"周公馆"+英文，完全不够
- 2024年翻新后可能有新说明牌，但内容未知且可能再次变更

### 解决方案：改为「法国梧桐密码」— 用树干标牌替代碑文

**核心概念：** 思南路的法国梧桐树有编号标牌（上海市园林绿化管理中心挂牌），每棵树有唯一编号。利用特定树木的编号序列作为密码。

**新谜题设计（3步）：**

**6-1 保持核心（找到正确位置）**
- 指令: "找到面朝周公馆方向的那排法国梧桐"
- 验证: 确认方向（朝东/朝西）
- 答案: `__FIELD__`

**6-2 改为「梧桐年轮密码」**
- 思南路梧桐树有绿化管理编号标牌（铁牌钉在树干上）
- 道具「窗格卡」改为「梧桐卡」: 一张印有6个方框的卡片
- 指令: "从周公馆门口向南，数到第3/5/7/9/11/13棵梧桐"
- 记录这6棵树的编号标牌末两位数字
- 6个数字按道具卡上的运算规则组合 = 密码
- 答案: `__FIELD__`（需实地记录具体树牌号）

**6-3 保持（地砖图案匹配）**
- 在特定梧桐树下观察地砖图案
- 匹配老地图图例
- 图案代号 = 位置标记

**为什么更好：**
- 梧桐树编号标牌是**市政基础设施**，极少变更（除非树被砍）
- 思南路梧桐是上海最著名的行道树，与法租界主题完美契合
- 不再需要精确对准（5mm容差问题消除）
- 操作简单: 走路、看牌、记数字
- 保留了"在思南路漫步"的浪漫体验

**备选方案（如梧桐树无编号牌）：**
改为「门牌号密码」——思南路/复兴中路的老洋房门牌号有特定规律（奇偶交替），用特定门牌号组合作为密码。门牌号永远不变。

## 6. 南京站6："洪武三十二年"谜题修正

### 研究结论
"洪武三十二年"作为年号**确实在历史上存在过**——朱棣夺位后废除建文年号，将建文元年(1399)改称洪武三十二年。但台城城墙上**没有查到这样一块修缮碑**的实物记录。

### 解决方案：保留"洪武三十二年"概念，改为「城砖铭文」而非「修缮碑」

**新谜题逻辑：**
台城段城墙上的城砖，和中华门一样带有铭文。某些砖上烧刻的年号可能因为特殊历史原因出现"洪武三十二年"——因为朱棣篡位后追改年号，城墙修补用砖按新规刻字。

**6-3 改为（城砖年号发现）：**
- 指令: "在台城城墙的内侧面，寻找一块与众不同的城砖"
- 道具: 纸尺（已有，用于6-1测量标石间距）+ 一张"砖铭年号对照表"
- 对照表列出: 洪武元年~洪武三十一年 = 正常 / 建文元年~建文四年 = 正常 / **洪武三十二年~三十五年 = 异常！**
- 问题: "为什么会有洪武三十二年？洪武明明只有三十一年。这意味着什么？"
- 答案选择（三选一）:
  A. 工匠刻错了
  B. 朱棣篡位后废除建文年号，追改为洪武
  C. 这是伪造的城砖
- 正确答案: B
- 碎片文字从答案关联的历史事件中提取

**为什么更好：**
- 不依赖一块特定的碑（碑可能不存在）
- 改为依赖**城砖铭文系统**（南京城墙砖铭文有17万块数字档案，极其丰富）
- "洪武三十二年"的历史知识点保留且更有教育性
- 从"找一块碑"改为"理解一段历史"，降低了对单一实物的依赖
- 即使台城段没有找到"洪武32年"砖，也可以作为**知识题**而非实物搜索题

**实地验证需求降低：** 改为知识选择题后，不再需要确认特定碑/砖的存在。

## 7. 上海站5：石库门"人脸"优化

### 研究结论
一大会址门楣确实有**矾红色盾形+卷草叶堆花图案+巴洛克涡卷**装饰，但没有文献记载"人脸"图案。属于pareidolia（面孔辨识偏误），可能因人而异。

### 解决方案：保留"在门楣装饰中发现隐藏图案"的概念，但改为更可靠的「盾形徽记」

**5-1 改为（盾形徽记辨识）：**
- 指令: "在兴业路的石库门建筑群中，找到门楣上带有**盾形装饰**的那扇门"
- 这是有明确文献记载的装饰元素（"矾红色盾形兼卷草叶堆花图案"）
- 问题: "盾形装饰两侧的涡卷（漩涡形花纹），左右各有几个？"
- 答案: `__FIELD__`（需实地计数）

**为什么更好：**
- "盾形装饰"有文献记载，确实存在
- "涡卷计数"是客观的，不依赖主观的pareidolia
- 保留了"在建筑装饰中发现细节"的观察力要求
- 涡卷数量是固定的建筑特征，不会随时间变化

## 10. 南京站3：解码链条逻辑审查与优化

### 现状（4步链条）
```
八股文提取 → 明远楼定位 → 状元表查找 → 号舍刻痕
```

### 问题分析
步骤本身不算多（4步比review中说的6步少），但问题在于：
1. 步骤2"明远楼定位"要求记录匾额文字+楼层数+朝向三个参数——如果任一记错，后续全断
2. 步骤3"状元表查找"要求用三元组坐标定位到特定年份——查表操作繁琐且容易看错行
3. 每步之间的逻辑跳转对普通玩家不够直觉

### 优化方案：保持4步但每步逻辑更自洽

**3-1 保持（八股文提取）— 但简化规则**
- 原规则: 取每段第N个字（N=段号）
- 优化: 改为"取每段的**首字**"——8段取8个首字，更直觉
- 8个首字组成一句指引，如"明远楼上望南方"（`__FIELD__`需设计）
- 这样不需要数字，降低出错概率

**3-2 优化（明远楼观察）— 减少参数**
- 原规则: 记录匾额文字+楼层数+朝向 = 三元组
- 优化: 只需回答一个问题——"明远楼匾额上写了什么？"
- 答案: `__FIELD__`（如"明远"两字）
- 匾额文字作为直接线索指向下一步（不再需要三元组坐标）

**3-3 优化（题名碑查找）— 更直接的关联**
- 原规则: 用三元组在状元表中定位
- 优化: 3-2的匾额文字中藏有一个年号线索（如"明远"→"明"→寻找明代最后一科）
- 或者更简单: 道具状元表上，在"明远"二字的笔画交叉处标注了一个年份
- 用该年份在题名碑上找到对应考生 → 该考生名字的笔画数 = 号舍编号

**3-4 保持（号舍刻痕）**
- 用号舍编号找到对应刻痕拓片
- 读出碎片文字

### 优化后的逻辑链
```
八股文首字 → "明远楼上望南方" → 到明远楼看匾额 → "明远"二字 → 道具卡上标注的年份 → 题名碑找人名 → 笔画数 → 号舍 → 刻痕
```

**关键改进：**
- 每一步只产生**一个**关键信息（不再有三元组）
- 步骤间过渡更自然（"明远"→"明代"→题名碑是自然联想）
- 提示系统可以在每步给出明确指引
- 保留了4步的深度，但每步出错概率大幅降低

.theme-suzhou {
