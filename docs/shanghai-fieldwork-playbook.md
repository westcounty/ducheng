## 第0站：解封（序章，在家完成）

> 不需要到任何地点。用道具和开场信息解出第一个目的地。

**屏幕显示：**

```
档案解封
━━━━━━━
解封
一份旧档案意外解封

烧毁的纸条残片:
  ██路██号
  ███的绿█
  █饭██

纸条背面:
  KHSLQJIDQGLDQ

日记封面便签:
  "同志：年份尾数，老规矩。"
```

**指令：**
> 档案袋里有一张烧毁大半的纸条，只剩几个残字。纸条背面有一行加密字母。
>
> 日记封面夹着一张便签："同志：年份尾数，老规矩。"
>
> 开场故事里提到了一个年份。用它的尾数做密码转盘偏移量，解密那行字母。解出的拼音和纸条残字拼在一起——你的第一个目的地是哪里？

**解谜过程：**
1. 开场故事提到"1943年秋" → 年份尾数 = **3**
2. 密码转盘偏移3位解密 `KHSLQJIDQGLDQ`：
   - K→H, H→E, S→P, L→I, Q→N, J→G, I→F, D→A, Q→N, G→D, L→I, D→A, Q→N
   - = **HEPINGFANDIAN** = 和平饭店（拼音）
3. 纸条残片"路""绿""饭"印证

**答案：** 和平饭店
**提示：**
1. 开场故事说的是"1943年秋"——年份尾数是几？
2. 用密码转盘偏移3位。K→H, H→E, S→P…解出来是一串拼音。
3. 解密结果是 HEPINGFANDIAN。纸条残字里有"路""绿""饭"——答案是「和平饭店」。

**通过后显示：** "目标确认——和平饭店。出发。" → [前往第一站]
