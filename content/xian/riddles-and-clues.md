# 谜语与线索

## 阶段间过渡线索

### 第一阶段 → 第二阶段
**般若说："阿史那在南边的大门等你。那个门洞像一只张开的嘴，但吞不下他。"**

### 第二阶段 → 第三阶段
**阿史那说："约翰那个洋和尚在城墙里头刻了一块碑。他说他的神和般若的佛都住在那块石头上。去看看。"**

### 第三阶段 → 第四阶段
**约翰说："颜真卿的字是用血写的。张怀远的字是用命写的。出了这里往西走，那条街上全是字。"**

### 第四阶段 → 第五阶段
**张怀远说："阿倍仲麻吕每天晚上听着鼓声数日子。他数了一万多个日子，还是没有回家。往北走，去听那面鼓。"**

### 第五阶段 → 第六阶段
**阿倍说："我回不了家，但伊本说他能在长安的西市找到家的味道。你往北走，那条街现在还在。"**

### 第六阶段 → 第七阶段
**伊本说："向南，找那座七层的塔。在塔下，我备好了酒。"**

## 各阶段内部线索概要

### 第一阶段 - 小雁塔/荐福寺
- 塔层计数（现存层数 vs 原15层）→ 缺失数+现存数参与运算
- 金代铁钟铭文 → 关键字/数字
- 密码梵文对照表解读

### 第二阶段 - 城墙南门
- 四正门名字配对（长乐/永宁/安定/安远 ↔ 字义）
- 瓮城俯视简图找错（3处）→ 网格坐标
- 城砖叠层（秦砖→汉土→唐夯→明砖）断面图验证

### 第三阶段 - 碑林
- 景教碑双语碑面分区 → 特定位置字符
- 书法辨体（颜/草/隶/篆）→ 匹配序列
- 颜真卿忠义叙事 → 碑刻拓片细节

### 第四阶段 - 书院门
- 残碑拓片解读（阳刻原理推断模糊字）
- 招牌密码（特定店铺偏旁/笔画提取）
- 关中书院定位 + 嵌入密码解读

### 第五阶段 - 钟鼓楼
- 鼓楼匾额"文武盛地""声闻于天" → 8字密码表
- 24节气知识推理 → 中间密码
- 坊市制度数字（开市/闭市鼓声数、宵禁时刻）
- 阿倍仲麻吕望乡和歌解读

### 第六阶段 - 回民街/清真大寺
- "胡"字食物溯源（胡椒/胡桃/胡麻/胡萝卜）→ 方位序列
- 康安的菜单配料数字 → 密码序列
- 清真大寺砖雕花中藏字辨识

### 第七阶段 - 大雁塔/不夜城（终章）
- 大雁塔层数+轮廓观察 → 排列参数1
- 玄奘铜像法器方位 → 排列参数2
- 碎片拼合 → 完整请帖
- 藏头揭示 → 此城容得下万邦
- 信封揭底 → 康安最后旅记 + 菜单 + 请帖全文
