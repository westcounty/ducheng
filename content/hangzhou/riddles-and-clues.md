# 谜语与线索

## 阶段间过渡线索

### 第一阶段 → 第二阶段
**沿白堤向西走。找那个用鹤当妻子的人——他住在湖中间的那座山上。**

### 第二阶段 → 第三阶段
**山中有泉，泉边有茶。她在学人间最苦的滋味。往西边的山里走——找十八棵树。**

### 第三阶段 → 第四阶段
**下山回到湖边。有个将军死了八百年，还有人替他生气。去看看那些跪着的人。**

### 第四阶段 → 第五阶段
**向南走上那条堤。一个写诗的胖子用泥巴做了一件比诗更伟大的事。数一数桥——每座桥都有名字。**

### 第五阶段 → 第六阶段
**听钟。净慈寺的钟声停了之后，继续听。你听到的不是声音——是回响。**

### 第六阶段 → 第七阶段
**出寺向南。最后一座塔。她在那里等你。**

## 各阶段内部线索概要

### 第一阶段 - 断桥
- "断"字拆解（米+刀+斤）→ 三组实景观察任务
- 断桥残雪石碑考据 → 选择正确解释
- 三位数密码ABC

### 第二阶段 - 孤山/西泠印社
- 林逋诗填空 → 缺字笔画数
- 残印拓片阴阳互补 → 方向指引
- 秋瑾碑辨识 → 密钥E

### 第三阶段 - 龙井村
- 御茶树碑文考证（三版本辨真伪）
- 茶谱密码（龙井四绝编码）
- 虎跑泉知识判断

### 第四阶段 - 岳王庙
- "青山有幸埋忠骨，白铁无辜铸佞臣"对联拆解
- 入口匾额+历史推理
- 岳飞书法风格辨认

### 第五阶段 - 苏堤六桥
- 六桥名序列采集（映波/锁澜/望山/压堤/东浦/跨虹）
- 望山桥结构观察（拱数、望柱数）
- 东坡肉文段密码（三点水字+桥名首字笔画差）

### 第六阶段 - 净慈寺
- 南屏晚钟知识辨错
- 济公禅机谜语
- 佛偈重组（四句排序→首字密钥）

### 第七阶段 - 雷峰塔（终章）
- 铜雕故事排序
- 碎片拼合 → 完整消息
- 藏头揭示 → 湖心有一人未走
- 信封揭底 → 第七页 + 许仙笔迹
