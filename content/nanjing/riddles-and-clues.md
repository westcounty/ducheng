# 谜语与线索

游戏各阶段完成后出现的转场线索，指引玩家前往下一阶段：

### 第一阶段 → 第二阶段
**巷深处，门楣上，木匠在百年前落了款。**

### 第二阶段 → 第三阶段
**天下英才尽入此门，但这扇门只有一米三宽。**

### 第三阶段 → 第四阶段
**秦淮水面上，倒着写的字才是正的。**

### 第四阶段 → 第五阶段
**四百八十寺，只有一座在鸡鸣。**

### 第五阶段 → 第六阶段
**登城头，向东走，柳树还在等那个写诗的人。**

### 第六阶段 → 第七阶段
**下城向北，湖边有一面墙记着全天下人的名字。**

## 各阶段内部线索概要

### 第一阶段 - 中华门城堡
- 城门正面结构观察（门洞数、层数）→ 组合密码A
- 用密码A查城砖索引卡 → 找指定铭文
- 铭文三栏各取首字 → 三字密钥

### 第二阶段 - 老门东
- 三条灯谜 → 猜出三条巷名（箍桶巷、剪子巷、边营）
- 辨识明/清/民国三种门楣 → 按时间排列门牌号 = 密码B
- 密码B对应老宅 → 找木匠落款

### 第三阶段 - 科举博物馆
- 八股残卷按格式提取8字 → 指引语
- 明远楼定位（匾额、层数、朝向）→ 检索坐标
- 状元表检索 → 姓名笔画数 = 密码C
- 密码C对应号舍 → 墙角刻字

### 第四阶段 - 秦淮河
- 取景框卡对准4种窗格 → 拼出对联出句
- 河畔石栏找对句 → 完整指令
- 手机前置摄像头翻转倒写文字 → 灯谜谜面
- 猜灯谜谜底

### 第五阶段 - 鸡鸣寺
- 佛塔数学：7层×8面=56，减去"首刹"编号
- 钟声编码（长响=1，短响=0）→ 二进制转十进制
- 定位还愿墙指定位置

### 第六阶段 - 台城城墙
- 纸尺丈量两块里程铭石间距 → 数据F
- 城墙上俯瞰+道具平视照片 → 双视角叠加密码G
- F+G定位修缮碑

### 第七阶段 - 玄武湖畔（终章）
- 碎片拼合 → 完整消息
- 藏头诗揭示 → 城巷有声音尚在
- 信封揭底 → 陆鸣远遗札 + 七人名单
